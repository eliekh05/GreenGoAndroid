Eco-friendly travel companion app — Android port of the iOS GreenGo app.

## Install the APK

1. Go to [Releases](https://github.com/eliekh05/GreenGoAndroid/releases) and download the latest `GreenGo.apk`
2. Transfer the APK to your Android phone
3. Open the file to install

### ⚠️ Installation Notes

**Auto-blocker (Newer Samsung on newer Android):**
Newer Samsung devices on newer Android versions have an "Auto Blocker" that prevents installing APKs from outside the Play Store.
To disable it: **Settings → Security → Auto Blocker → turn Off**
To revert it: **Settings → Security → Auto Blocker → turn On**

**Google Play Protect warning:**
Because this app is not on the Play Store, Play Protect will show a warning.
To install: tap **More details** → **Install anyway** or **Install without scanning**


**PIN / biometric prompt:**
Devices with PIN or fingerprint enabled needs to confirm installation of sideloaded APKs. This is normal — enter your PIN to proceed.
